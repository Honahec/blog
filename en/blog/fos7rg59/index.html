Modular Arithmetic with Rational Numbers

Problem Description

P2613 【Template】Modular Arithmetic with Rational Numbers

The problem is very simple and easy to understand

Given a rational number c=abc=\frac{a}{b}c=ba​, find the value of c mod 19260817c\bmod19260817cmod19260817.

Approach

When looking at this problem:

  • First glance: Seems pretty simple
  • Second glance: a,b≤1010001a,b\leq10^{10001}a,b≤1010001?!!
  • Third glance: What the heck is this?!!

To solve this problem, let me introduce something new (mainly because I don't know exgcd (Extended Euclidean Algorithm))

Modular Inverse

P is a prime number

Let b′b'b′ be the modular inverse of bbb, then we have b×b′≡1 (mod P)b\times b'\equiv1\,(mod\,P)b×b′≡1(modP)

We have ab≡ab×1≡ab×(b×b′)≡a×b′ (mod P)\frac{a}{b}\equiv\frac{a}{b}\times1\equiv\frac{a}{b}\times(b\times b')\equiv a\times b'\,(mod\,P)ba​≡ba​×1≡ba​×(b×b′)≡a×b′(modP)

Clearly we can see that the value of b′b'b′ is 1b\frac{1}{b}b1​

But dividing by bbb is the same as multiplying by 1b\frac{1}{b}b1​, so what do we do?

Fermat's Little Theorem to the Rescue

Theorem: If ppp is a prime number and integer aaa is not a multiple of ppp, then ap−1≡1 (mod p)a^{p-1}\equiv1\,(mod\,p)ap−1≡1(modp)

Proof

In the proof section, ppp is a prime number

First Proof

First, let's consider a prerequisite theorem:

If gcd(c,p)=1gcd(c,p)=1gcd(c,p)=1 and ac≡bc (mod p)ac\equiv bc\,(mod\,p)ac≡bc(modp), then a≡b (mod p)a\equiv b\,(mod\,p)a≡b(modp)

Proof:

∵ac≡bc (mod p)∴(a−b)c≡0 (mod p)∴(a−b)c is an integer multiple of pAlso∵gcd(c,p)=1∴a−b≡0 (mod p)i.e. a≡b (mod p)\begin{align} &\because ac\equiv bc\,(mod\,p)\\ &\therefore (a-b)c\equiv 0\,(mod\,p)\\ &\therefore (a-b)c\,\text{is an integer multiple of}\,p\\ \text{Also}&\because gcd(c,p)=1\\ &\therefore a-b\equiv0\,(mod\,p)\\ &\text{i.e.}\,a\equiv b\,(mod\,p) \end{align} Also​∵ac≡bc(modp)∴(a−b)c≡0(modp)∴(a−b)cis an integer multiple ofp∵gcd(c,p)=1∴a−b≡0(modp)i.e.a≡b(modp)​​

Q.E.D.

Second Proof

Now let's get to the main point. Suppose there's a positive integer a(a<p)a(a<p)a(a<p) satisfying gcd(a,p)=1gcd(a,p)=1gcd(a,p)=1, then multiplying aaa by 1∼p−11\thicksim p-11∼p−1 forms a complete residue system mod pmod\,pmodp

Proof:

Assume there exist xa≡ya (mod p),and x≠y∵gcd(a,p)=1∴the equation holds if and only if x≡y (mod p)Also∵x,y∈[1,p−1]∴x≡y (mod p)if and only if x=y,which contradicts the given condition∴the assumption is false, the original proposition holds\begin{align} &\text{Assume there exist}\,xa\equiv ya\,(mod\,p),\text{and}\,x\neq y\\ &\because gcd(a,p)=1\\ &\therefore \text{the equation holds if and only if}\,x\equiv y\,(mod\,p)\\ \text{Also}&\because x,y\in [1,p-1]\\ &\therefore x\equiv y\,(mod\,p)\text{if and only if}\,x=y,\text{which contradicts the given condition}\\ &\therefore \text{the assumption is false, the original proposition holds} \end{align} Also​Assume there existxa≡ya(modp),andx=y∵gcd(a,p)=1∴the equation holds if and only ifx≡y(modp)∵x,y∈[1,p−1]∴x≡y(modp)if and only ifx=y,which contradicts the given condition∴the assumption is false, the original proposition holds​​

Q.E.D.

Third Proof

Now let's prove ap−1≡1 (mod p)a^{p-1}\equiv1\,(mod\,p)ap−1≡1(modp)

Proof:

∵1,2,⋯ ,p−1is a complete residue system mod p∴we have 1×2×⋯×p−1≡a×2a×⋯×(p−1)a (mod p)i.e. (p−1)!≡(p−1)!×ap−1 (mod p)Also∵pis prime∴gcd((p−1)!,p)=1∴ap−1≡1 (mod p)\begin{align} &\because 1,2,\cdots,p-1\text{is a complete residue system}\,mod\,p\\ &\therefore\text{we have}\,1\times2\times\cdots\times p-1\equiv a\times2a\times\cdots\times (p-1)a\,(mod\,p)\\ &\text{i.e.}\,(p-1)!\equiv (p-1)!\times a^{p-1}\,(mod\,p)\\ \text{Also}&\because p\text{is prime}\\ &\therefore gcd((p-1)!,p)=1\\ &\therefore a^{p-1}\equiv1\,(mod\,p) \end{align} Also​∵1,2,⋯,p−1is a complete residue systemmodp∴we have1×2×⋯×p−1≡a×2a×⋯×(p−1)a(modp)i.e.(p−1)!≡(p−1)!×ap−1(modp)∵pis prime∴gcd((p−1)!,p)=1∴ap−1≡1(modp)​​

Q.E.D.

Back to the Problem

According to Fermat's Little Theorem, clearly b′=bP−2b'=b^{P-2}b′=bP−2

What? Not obvious?

∵b′×b≡1 (mod P)bP−1≡1 (mod P)∴b′×b≡bP−1 (mod P)∴b′=bP−2\begin{align} \because b'\times b&\equiv1\,(mod\,P)\\ b^{P-1}&\equiv1\,(mod\,P)\\ \therefore b'\times b&\equiv b^{P-1}\,(mod\,P)\\ \therefore b'&=b^{P-2} \end{align} ∵b′×bbP−1∴b′×b∴b′​≡1(modP)≡1(modP)≡bP−1(modP)=bP−2​​

So ab=a×bP−2\frac{a}{b}=a\times b^{P-2}ba​=a×bP−2

Then we can easily solve this problem using fast exponentiation (pspsps: although fast exponentiation means we don't need arbitrary precision arithmetic, we still need long long)

Code

#include<cstdio>

const int MOD = 19260817;

inline long long read() {
	long long X = 0, flag = 1; char ch = getchar();
	while(ch < '0' || ch > '9') {if(ch == '-') flag = 0; ch=getchar();}
	while(ch >= '0' && ch <= '9') {X = (X * 10 + (ch ^ 48)) % MOD; ch = getchar();}
	return flag ? X : -X;
}

long long a, b;

long long QuickPow(long long x, long long y) {
	long long ans = 1, k = MOD - 2;
	while(k) {
		if(k & 1) ans = ans * x % MOD;
		x = x * x % MOD;
		k >>= 1;
	}
	return ans * y % MOD;
}//Not recommended to write fast exponentiation this way, better to use a template

int main() {
	a = read(); b = read();
	if(b) printf("%lld\n", QuickPow(b, a));
	else printf("Angry!\n");
	return 0;
}
