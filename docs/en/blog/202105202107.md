# Whack-a-Mole

## Problem Link

[P2285 Whack-a-Mole](https://www.luogu.com.cn/problem/P2285)

## Approach

Obviously, this is a $dp$ problem, because the tag already tells us (not really

How to do $dp$?

Chessboard? Three dimensions will definitely explode, ~~this noob doesn't know how to do two dimensions~~

Time? ~~Even more ridiculous~~

So we can only work with the moles themselves

Let $dp_i$ be the maximum number of moles that can be whacked up to the $i$-th mole

Since we need to wait for the moles to come to us, initialize $dp_i=1$

If the [Manhattan distance](https://baike.baidu.com/item/曼哈顿距离/743092?fr=aladdin) between two moles is less than the time difference

The transition equation is $dp_i=min(dp[i],dp[j]+1)$

## Code

```c++
#include<cstdio>
#include<cmath>
#define fo(i,x,y) for(register int i=x;i<=y;++i)
#define go(i,x,y) for(register int i=x;i>=y;--i)
#define INF 0x3f3f3f3f
#define il inline
#define ll long long
#define ull unsigned long long
using namespace std;

const int N = 1005, M = 10005;
int n, m, dp[M], ans;
struct NODE {
	int time, x, y;
}node[M];

il int max(const int &a, const int &b) {return a > b ? a : b;}

int main() {
	scanf("%d%d", &n, &m);
	fo(i, 1, m) scanf("%d%d%d", &node[i].time, &node[i].x, &node[i].y), dp[i] = 1;
	fo(i, 1, m) {
		fo(j, 1, i - 1) {
			if(abs(node[i].x - node[j].x) + abs(node[i].y - node[j].y) <= abs(node[i].time - node[j].time)) {
				dp[i] = max(dp[i], dp[j] + 1);
			}
		}
	}
	fo(i, 1, m) ans = max(ans, dp[i]);
	printf("%d\n", ans);
	return 0;
}
```
