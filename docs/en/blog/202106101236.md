# Modular Arithmetic with Rational Numbers

## Problem Description

[P2613 【Template】Modular Arithmetic with Rational Numbers](https://www.luogu.com.cn/problem/P2613)

### The problem is very simple and easy to understand

Given a rational number $c=\frac{a}{b}$, find the value of $c\bmod19260817$.

## Approach

When looking at this problem:

- First glance: Seems pretty simple
- Second glance: $a,b\leq10^{10001}$?!!
- Third glance: What the heck is this?!!

To solve this problem, let me introduce something new ~~(mainly because I don't know `exgcd` (Extended Euclidean Algorithm))~~

### Modular Inverse

> P is a prime number

Let $b'$ be the modular inverse of $b$, then we have $b\times b'\equiv1\,(mod\,P)$

We have $\frac{a}{b}\equiv\frac{a}{b}\times1\equiv\frac{a}{b}\times(b\times b')\equiv a\times b'\,(mod\,P)$

Clearly we can see that the value of $b'$ is $\frac{1}{b}$

But dividing by $b$ is the same as multiplying by $\frac{1}{b}$, so what do we do?

### Fermat's Little Theorem to the Rescue

**Theorem: If $p$ is a prime number and integer $a$ is not a multiple of $p$, then $a^{p-1}\equiv1\,(mod\,p)$**

#### Proof

**You can choose to [skip](#back-to-the-problem) this section**

> In the proof section, $p$ is a prime number

##### First Proof

First, let's consider a prerequisite theorem:

If $gcd(c,p)=1$ and $ac\equiv bc\,(mod\,p)$, then $a\equiv b\,(mod\,p)$

**Proof:**

$$
\begin{align}
&\because ac\equiv bc\,(mod\,p)\\
&\therefore (a-b)c\equiv 0\,(mod\,p)\\
&\therefore (a-b)c\,\text{is an integer multiple of}\,p\\
\text{Also}&\because gcd(c,p)=1\\
&\therefore a-b\equiv0\,(mod\,p)\\
&\text{i.e.}\,a\equiv b\,(mod\,p)
\end{align}
$$

**Q.E.D.**

##### Second Proof

Now let's get to the main point. Suppose there's a positive integer $a(a<p)$ satisfying $gcd(a,p)=1$, then multiplying $a$ by $1\thicksim p-1$ forms a [complete residue system](https://baike.baidu.com/item/%E5%AE%8C%E5%85%A8%E5%89%A9%E4%BD%99%E7%B3%BB) $mod\,p$

**Proof:**

$$
\begin{align}
&\text{Assume there exist}\,xa\equiv ya\,(mod\,p),\text{and}\,x\neq y\\
&\because gcd(a,p)=1\\
&\therefore \text{the equation holds if and only if}\,x\equiv y\,(mod\,p)\\
\text{Also}&\because x,y\in [1,p-1]\\
&\therefore x\equiv y\,(mod\,p)\text{if and only if}\,x=y,\text{which contradicts the given condition}\\
&\therefore \text{the assumption is false, the original proposition holds}
\end{align}
$$

**Q.E.D.**

##### Third Proof

Now let's prove $a^{p-1}\equiv1\,(mod\,p)$

**Proof:**

$$
\begin{align}
&\because 1,2,\cdots,p-1\text{is a complete residue system}\,mod\,p\\
&\therefore\text{we have}\,1\times2\times\cdots\times p-1\equiv a\times2a\times\cdots\times (p-1)a\,(mod\,p)\\
&\text{i.e.}\,(p-1)!\equiv (p-1)!\times a^{p-1}\,(mod\,p)\\
\text{Also}&\because p\text{is prime}\\
&\therefore gcd((p-1)!,p)=1\\
&\therefore a^{p-1}\equiv1\,(mod\,p)
\end{align}
$$

**Q.E.D.**

#### Back to the Problem

According to Fermat's Little Theorem, clearly $b'=b^{P-2}$

What? Not obvious?

$$
\begin{align}
\because b'\times b&\equiv1\,(mod\,P)\\
b^{P-1}&\equiv1\,(mod\,P)\\
\therefore b'\times b&\equiv b^{P-1}\,(mod\,P)\\
\therefore b'&=b^{P-2}
\end{align}
$$

So $\frac{a}{b}=a\times b^{P-2}$

Then we can easily solve this problem using fast exponentiation ($ps$: although fast exponentiation means we don't need arbitrary precision arithmetic, we still need `long long`)

## Code

```c++
#include<cstdio>

const int MOD = 19260817;

inline long long read() {
	long long X = 0, flag = 1; char ch = getchar();
	while(ch < '0' || ch > '9') {if(ch == '-') flag = 0; ch=getchar();}
	while(ch >= '0' && ch <= '9') {X = (X * 10 + (ch ^ 48)) % MOD; ch = getchar();}
	return flag ? X : -X;
}

long long a, b;

long long QuickPow(long long x, long long y) {
	long long ans = 1, k = MOD - 2;
	while(k) {
		if(k & 1) ans = ans * x % MOD;
		x = x * x % MOD;
		k >>= 1;
	}
	return ans * y % MOD;
}//Not recommended to write fast exponentiation this way, better to use a template

int main() {
	a = read(); b = read();
	if(b) printf("%lld\n", QuickPow(b, a));
	else printf("Angry!\n");
	return 0;
}
```
