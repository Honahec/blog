# Matrix

## Matrix Definition

An $n\times m$ matrix is a rectangular array of elements arranged in $n$ rows and $m$ columns. The elements in a matrix can be numbers, symbols, or mathematical expressions.

A table in the form of $\left(\begin{matrix}a&b\\c&d\end{matrix}\right)$ is called a **second-order matrix**, which consists of two rows and two columns, where $a,b,c,d$ are called the elements of this matrix.

An ordered pair in the form of $\left(\begin{matrix}x_1\\x_2\end{matrix}\right)$ is called a **column vector**.

## Matrix Addition

Matrix addition is very simple.

Yes, it's exactly what you think.

$$
\left(
\begin{matrix}
a_{1,1}&a_{1,2}&\cdots&a_{1,m}\\a_{2,1}&a_{2,1}&\cdots&a_{2,m}\\\vdots&\vdots&&\vdots\\a_{n,1}&a_{n,2}&\cdots&a_{n,m}
\end{matrix}
\right)\pm
\left(
\begin{matrix}
b_{1,1}&b_{1,2}&\cdots&b_{1,m}\\b_{2,1}&b_{2,1}&\cdots&b_{2,m}\\\vdots&\vdots&&\vdots\\b_{n,1}&b_{n,2}&\cdots&b_{n,m}
\end{matrix}
\right)=
\left(
\begin{matrix}
a_{1,1}\pm b_{1,1}&a_{1,2}\pm b_{1,2}&\cdots&a_{1,m}\pm b_{1,m}\\a_{2,1}\pm b_{2,1}&a_{2,1}\pm b_{2,2}&\cdots&a_{2,m}\pm b_{2, m}\\\vdots&\vdots&&\vdots\\a_{n,1}\pm b_{n,1}&a_{n,1}\pm b_{n,2}&\cdots&a_{n,m}\pm b_{n,m}
\end{matrix}
\right)
$$

Note: Two matrices must be **matrices of the same type** (i.e., both must be $n\times m$ matrices) to be added.

Moreover, matrix addition and subtraction **satisfy** the commutative and associative laws (i.e., $A+B=B+A$, $(A+B)+C=A+(B+C)$).

## Matrix Multiplication

Matrix multiplication is **slightly** different.

Well, really just a little different.

If matrix $A$ has size $n\times m$, and matrix $B$ has size $m\times s$, their product is matrix $C$, and matrix $C$ must have size $n\times s$.

Matrix multiplication follows $C=\sum_{k=1}^{m}a_{i,k}b_{k,j}$.

Well, just a little.

### An Example

Let

$$
\begin{align}
A&=\left(\begin{matrix}a&b\\c&d\end{matrix}\right)\\\\
B&=\left(\begin{matrix}x_1\\x_2\end{matrix}\right)\\
\end{align}
$$

Then

$$
\begin{align}
C=AB&=\left(\begin{matrix}ax_1+bx_2\\cx_1+dx_2\end{matrix}\right)\\
\end{align}
$$

This is called the product of the second-order matrix $A$ and the plane vector $B$, denoted as $AB=C$.

## Using Matrices to Calculate the Fibonacci Sequence

As we all know, starting from the third term of the Fibonacci sequence, each term is the sum of the previous two terms.

That is, $F_n=F_{n-1}+F_{n-2},n\geq3$.

In particular, $F_0=0,F_1=F_2=1$.

Write two adjacent terms in the Fibonacci sequence ($F_n$ and $F_{n-1}$) as a $2\times1$ matrix:

$$
\begin{align}
&=\left(
\begin{matrix}
F_n\\ F_{n-1}
\end{matrix}
\right)\\\\
&=\left(
\begin{matrix}
F_{n-1}+F_{n-2}\\ F_{n-1}
\end{matrix}
\right)\\\\
&=\left(
\begin{matrix}
1\times F_{n-1}+1\times F_{n-2}\\ 1\times F_{n-1}+0\times F_{n-2}
\end{matrix}
\right)\\\\
&=\left(
\begin{matrix}
1&1\\ 1&0
\end{matrix}
\right)
\times \left(
\begin{matrix}
F_{n-1}\\ F_{n-2}
\end{matrix}
\right)\\\\
&=\left(
\begin{matrix}
1&1\\ 1&0
\end{matrix}
\right)^{n-1}
\times \left(
\begin{matrix}
F_1\\ F_0
\end{matrix}
\right)\\\\
&=\left(
\begin{matrix}
1&1\\ 1&0
\end{matrix}
\right)^{n-1}
\times \left(
\begin{matrix}
1\\ 0
\end{matrix}
\right)
\end{align}
$$

Finding $F_n$ is equivalent to finding the $(n-1)$-th power of a second-order matrix, with the result being the first row element of the matrix.

The problem is transformed into finding the $n$-th power of a second-order matrix.

### Finding the $N$-th Power of a Second-Order Matrix

Here we can review [Matrix Multiplication](#matrix-multiplication).

**Assume we are calculating the $N$-th power of matrix $A$**

#### Method 1

Second-order matrix multiplication satisfies the associative law.

Let $A,B,C$ all be arbitrary second-order matrices.

Then $A(BC)=(AB)C$.

> Proof not provided here

Let $n=N\div 2$ (result rounded down).

If $N\in 2k$, then $A^N=A^n\times A^n$.

If $N\in 2k+1$, then $A^N=A^n\times A^n\times A$.

> This can reduce the number of calculations; think about why on your own

#### Method 2

Taking the calculation of $A^6$ as an example:

For instance, $6_{(10)}=110_{(2)}$.

Then $A^6=A^4\times A^2$.

![Binary bits](https://z3.ax1x.com/2021/06/15/2qY2uV.png)

The above figure shows the relationship between binary and power exponents.

When the binary bit is 1, multiplication is needed; when it's 0, multiplication is not needed.

Another example: $7_{(10)}=111_{(2)}$.

Then $A^7=A^4\times A^2\times A^1$.

First, let's casually write some code for fast exponentiation (I'm sure everyone knows how to write it):

```c++
int qpow(int a, int k) {
    int res = 1;
    while(k) {
        if(a & 1) res *= a;
        k >>= 1;
        a *= a;
    }
    return res;
}
```

So how exactly do we use matrices to calculate the Fibonacci sequence?

### Code

```c++
#include<cstdio>
#include<cstring>

struct Matrix {
	long long fib[2][2];
	Matrix() {memset(fib, 0, sizeof fib);}
};

Matrix qmul(Matrix &a, Matrix &b) {
	Matrix c;
	for(int i = 0; i < 2; ++i) {
		for(int j = 0; j < 2; ++j) {
			for(int k = 0; k < 2; ++k) {
				c.fib[i][j] += a.fib[i][k] * b.fib[k][j];
			}
		}
	}
	return c;
}

Matrix qpow(Matrix a, long long k) {
	Matrix ans;
	ans.fib[0][0] = ans.fib[1][1] = 1;
	while(k) {
		if(k & 1) ans = qmul(ans, a);
		a = qmul(a, a);
		k >>= 1;
	}
	return ans;
}

int main() {
	long long n;
	Matrix a, b;
	scanf("%lld", &n);
	a.fib[0][0] = a.fib[0][1] = a.fib[1][0] = 1;
	b = qpow(a, n);
	printf("%lld\n", b.fib[0][1]);
	return 0;
}
```
